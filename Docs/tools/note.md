# note

# Description

keep notes

## Usage
```
Usage: pt note [-h] [-l] [-d DELETE] [-r] [note ...]

keep notes

positional arguments:
  note                  note name

options:
  -h, --help            show this help message and exit
  -l, --list            list all notes
  -d DELETE, --delete DELETE
                        delete a note
  -r, --replace         replace note
```
